# WalkingMileageService

### Walking Mileage Service Description
- 각자 공부하고자 하는 기술을 적용하여 완성해보는 토이 프로젝트
- 2주에 한번 프로젝트 진행 사항 공유 및 회의 진행

---

### 서비스 설명
- 사용자의 핸드폰에서 측정되는 걸음 수를 기반으로 각종 챌린지에 참가하여 혜택을 누릴 수 있는 서비스
- 각종 건강 정보 제공
- 커뮤니티를 기능 제공
- 추가 중....
